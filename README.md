# -assignment-html-css

## Giới thiệu dự án

Dự án "Ứng dụng Quản Lý Quán Cafe" được xây dựng nhằm hỗ trợ quản lý hiệu quả các hoạt động trong quán cafe như:

- Quản lý menu đồ uống
- Quản lý bàn
- Quản lý nhân viên
- Gọi món
- Thanh toán hóa đơn
- Thống kê doanh thu

## Mục tiêu

- Giảm sai sót trong quá trình gọi món
- Tăng tốc độ thanh toán
- Quản lý doanh thu hiệu quả
- Quản lý nhân viên và bàn dễ dàng
